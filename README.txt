# 咖啡店罗盘 · 新店选址与业绩预测工具

## 项目简介
一款基于机器学习的新店选址辅助工具，输入门店基本信息和租金条款，即可预测未来多年的日均交易量、年收入和年利润，并给出投资汇报分析。适用于咖啡、餐饮零售等行业门店开发人员快速评估点址

##主要功能
1. 门店信息录入：省份、城市、商圈、子商圈、门店类型、面积
2. 租赁条款设置：租期（仅支持整数）、首年年租金、租金年递增率
3. 逐年预测：输出日交易量、年收入、年利润、盈亏平衡交易量
4. 成本构成明细：随机预估原材料、人工、水电杂费、折旧成本
5. 趋势图和表格：展示多年业绩裱花，负值字体为红色
6. 结果导出：支持CSV 文件下载

##技术栈
1. 前端/部署：Streamlit cloud
2. 后端：python 3.14
3. 机器学习：LightGBM, Scikit_learn
4. 数据处理：pandas, numpy
5. 版本控制：GitHub

##模型构建
数据来源：某咖啡连锁品牌历史经营数据
特征工程：
    ——固定特征：省份、城市、城市等级（Tier）、商圈、子商圈、设计类型、面积
    ——目标编码：城市平均 SPC、城市平均客单价（AT）
    ——时间特征：年份（用于趋势）
模型设计：
   ——日均交易量 预测：LightGBM 回归
   ——年收入 预测：LightGBM 回归
   ——利润 计算：由 NetRevenue、租金、成本率公式推导（无需单独训练）
性能指标：
    ——日均交易量模型R²：0.848
    ——年收入模型R²：0.877
模型文件：'adt_model.pkl', 'net_model.pkl', 'city_spc_mean.pkl', 'city_at_pkl'

#注意事项
1. 所有门店数据已处理，仅保留业务特征，不涉及个人隐私
2. 模型局限性：预测结果基于历史数据，未来市场变化可能导致偏差，建议结合实地调研使用
3. 成本率假设：成本率为随机范围估算（原材料 20-30%、人工 18-25%、水电 3-5%、折旧 5-8%），实际成本请参考企业财务数据
4. 投资回收期：基于预测年利润累加，未考虑资金时间价值，仅供快速参考

#模型可提高
1. 如果能增加宏观指标（GDP/人口/GIS 人流数据）可以大大提高准确性，而且能作为时间趋势特征；
2. 多门店对比分析
3. 引入地图信息可视化，结合API 圈选商圈划分人流量，用人流量*捕获率计算日均交易量会更准确
4. 成本率自定义，可根据公司内部实际情况调整
